На удивление, jupyter использовал системный python, а мне нужна была его локальная версия из .venv виртуального окружения моего проекта.
python -m ipykernel install --user --name=mlp-venv --display-name "Python (ML-P venv)"
создала специализированный kernel в котором сохранились предустановленные библиотеки.
